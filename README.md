# STT_812_project
# Lung Cancer Survival Analysis

This project analyzes lung cancer survival data using statistical modeling and visualization in R.

## Overview

The goal of this project is to:
- Explore survival trends in lung cancer patients
- Fit statistical models to understand key predictors
- Visualize relationships between variables and survival outcomes
